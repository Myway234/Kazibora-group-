# KaziBora Group - Premium Website

A Next.js website for KaziBora Group, a Pan East African diversified holding company specializing in Hospitality, VIP Concierge Services, and Commodities.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Myway234/Kazibora-group-.git
cd Kazibora-group-
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Deployment on Vercel

1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Import this GitHub repository
4. Click "Deploy"
5. Get your live shareable URL!
